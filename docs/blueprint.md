# **App Name**: Profolio

## Core Features:

- Project Case Studies: Prominently display project case studies with images, descriptions, and key outcomes.
- About Me Section: Engaging 'About Me' page detailing skills, projects, experience, and personal interests.
- Contact Form: A clear 'Contact' form to allow users to send messages and inquiries.
- Image Gallery: Image gallery, separate from the case studies, where images may be freely uploaded. AI-powered tool that may flag inappropriate content.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to evoke professionalism and trust.
- Background color: Soft gray (#F2F4F7) to maintain a clean and minimalist look.
- Accent color: Subtle gray-blue (#A9B7C4) for highlights and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined feel, ensuring readability and a clean aesthetic.
- Use simple, geometric icons for navigation and key features.
- Clean, grid-based layout to organize content logically.
- Subtle transitions and hover effects to enhance user experience.
